ШАГ 1.  УСТАНОВИТЬ ЗАВИСИМОСТИ (один раз)
─────────────────────────────────────────
Откройте cmd в папке проекта и выполните:

    pip install -r requirements.txt


ШАГ 2.  ЗАДАТЬ API-КЛЮЧ (один раз)
─────────────────────────────────────────
Скопируйте .env.example, переименуйте копию в .env
Откройте .env в Блокноте и впишите:

    OPENAI_API_KEY=sk-ваш_настоящий_ключ
    OPENAI_BASE_URL=https://api.proxyapi.ru/openai/v1  (или другой посредник)
    OPENAI_MODEL=gpt-4o


ШАГ 3.  ПРОВЕРИТЬ, ЧТО ВСЁ УСТАНОВЛЕНО
─────────────────────────────────────────
    python test_smoke.py

Должно вывести: "🎉 OK — приложение работает"


ШАГ 4.  ЗАПУСТИТЬ ПРИЛОЖЕНИЕ
─────────────────────────────────────────
    streamlit run app.py

Откроется браузер. Дальше — по интерфейсу.

Подробная инструкция — в README.md
